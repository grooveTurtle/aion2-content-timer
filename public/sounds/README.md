# 알람 사운드 파일

이 폴더에 알람 사운드 파일을 추가하세요.

필요한 파일:
- `default.mp3` - 기본 사운드
- `chime.mp3` - 차임벨
- `bell.mp3` - 종소리
- `beep.mp3` - 비프음

## 무료 사운드 리소스

다음 사이트에서 무료 알람 사운드를 다운로드할 수 있습니다:

- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

또는 간단한 테스트용으로 브라우저의 기본 알림음을 사용할 수도 있습니다.
